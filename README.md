# TechWonders - Blog com Angular

## Introdução

Este é um projeto de aplicação web simples, desenvolvido com o framework **Angular**, no formato de blog. Possui duas rotas principais: uma para a página inicial e outra para a visualização de um artigo completo. O layout foi inspirado neste [design no Dribbble](https://dribbble.com/shots/18089191-Blog-Layout), criado por Dwinawan.

## Sobre o Projeto

### Instalação

```bash
# Clone o repositório
git clone https://github.com/angular-blog.git

# Acesse a pasta do projeto
cd angular-blog

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
ng serve

# A aplicação será iniciada na porta 4200
```

## Funcionalidades

- O blog é dividido em quatro seções principais: **posts do blog**, **UX/UI design**, **tutoriais de desenvolvimento web** e **sobre o blog**;
- Navegação fluida pela header com ancoragem para as seções da página inicial;
- Cards com imagem, título e data de postagem. Alguns incluem descrição e botão "Leia mais";
- Todos os cards são clicáveis (imagem, título e botão) e redirecionam para a página do artigo;
- Cada artigo possui: título, data, categoria, imagem de destaque e conteúdo completo;
- Página de artigo possui link de retorno para a home.

## Destaques do Projeto

- **Design Responsivo:** Layout adaptado para diferentes tamanhos de tela, proporcionando boa experiência tanto no desktop quanto em dispositivos móveis;
- **Navegação Intuitiva:** Menu de navegação fixo com links que levam diretamente para as seções da página inicial, facilitando o acesso rápido;
- **Uso de Angular:** Implementação prática do Angular com rotas, componentes reutilizáveis e gerenciamento simples do estado da aplicação;
- **Conteúdo Dinâmico:** Exibição dinâmica dos posts com dados mockados, permitindo fácil expansão para conteúdos reais futuros;
- **Integração Visual:** Utilização de imagens de alta qualidade e elementos visuais que valorizam a leitura e o interesse dos usuários;
- **SEO Básico:** Estrutura HTML semântica que ajuda na indexação e na acessibilidade do blog.
